Frequently Asked Questions

Home Page

1. How accurate are your weather forecasts?

We take forecasting very seriously at WeatherWiz and strive to be as accurate as possible. Our team actively monitors all of the available weather data and uses it to improve the forecasts that we deliver to you.

2. In what countries are you available?

WeatherWiz is available in the almost all popular places around the world, but we are actively working to expand our coverage to include all cities. Stay tuned!

3. Where do you get your data?

WeatherWiz uses a API from Google and darksky.net, which are aggregated together statistically to provide the most accurate forecast possible for a given location.

Saved Cities Page

1. Do I need to Sign up / Log in to view the Saved Cities Page?

Yes, the user needs to create an account through the sign up page and login with the credentials to access the Saved Cities Page. If the user visits the website as a guest then the Saved cities button on the Home page menu is disabled.

2. How many cities can i add to my Saved Cities Page?

A user who has signed up for the website can add up to six cities to the Saved Cities page. If the user has less than six cities added to the Saved Cities page, a tile with a plus sign will be available to add a new city.

3. Can I add cities from all over the world to my Saved Cities Page?

A user is allowed to add any city from all over the world without any restriction on user’s current location and country. The user can click on the plus tile and search for any city from all over the globe to be saved.

My Account Page

1. What information is saved in the User Profile of My Account Page?

The website stores the basic information about a registered user which includes the name of a user in addition to email address and mobile number. The website also stores the user password and any of this information can be updated by the user at any time.

2. How can I update my information on the My Account Page?

The user needs to login using the credentials to visit the User Profile section of My Account Page where the user has an option of updating name, phone number, email address and password. As soon as the user clicks on Save button, all the new details are saved.

3. Will my mobile number will be stored only in WeatherWiz database or it can be shared with any other third party organization?

The mobile number of the user is only stored in the WeatherWiz database for the future use case of sending out mobile notifications. WeatherWiz does not share any personal information of a user including phone numbers with any third party. Hence, the data of user is secure and safe.
